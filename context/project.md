# BioARC — Project Identity

**BioARC Specimen Collection Survey** — a geo-enriched field data collection web app.
Field researchers submit animal specimen records (species, location, photos, environmental covariates) via a 5-step card UI. Data archives to KoboToolbox via REST API.

## Background
- PI / supervisor at University of Michigan (jiayuanj@umich.edu)
- Demo-first approach: show supervisor working UI, then decide on auth / dashboard / server
- Longer term: could expand to mobile-first offline field tool
- Related platform studied: KoboToolbox, iNaturalist, Epicollect5, SMART
